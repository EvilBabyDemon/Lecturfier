# Lecturfier

[![D-INFK 2020](readme_images/d_trav.gif)](https://discord.gg/eth-dinfk-2020)

Lecturfier is a Discord Bot that helps out in tons of ways for the ETH D-INFK 2020 Discord Server.

  - Pings users on website updates
  - Tracks stats
  - Stores quotes
  - and much much more
